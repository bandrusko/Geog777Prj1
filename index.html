How this works

Every time you change k and click "Run Analysis," the app re-reads the well_nitrate points and re-runs inverse distance weighting from scratch to build the nitrate surface, then re-samples that surface inside every cancer zone and re-runs the regression. Results are only cached after they've been computed once, so re-selecting a k you've already run is instant, but the first run of any k is always a live calculation from the source data.

The K Comparison layer follows the same caching rules — if either k value was already computed, comparing them requires no additional IDW work.
